# Glossary

## Protocols

- **TLS**: Transport Layer Security (TLS) is a widely used cryptographic protocol designed to provide secure, encrypted communication over computer networks, primarily for Internet traffic. It succeeds the outdated SSL protocol, ensuring data integrity, authentication, and privacy between web browsers, servers, and other applications.
- **mTLS**: Mutual TLS (mTLS) is a security protocol requiring both client and server to authenticate each other via digital certificates, ensuring trust in both directions before establishing a connection. Unlike standard TLS, which only authenticates the server, mTLS provides superior security for APIs, IoT, and microservices.
- **ICS**- Industrial Control Systems (ICS) are specialized, automated electronic systems used to manage industrial processes, critical infrastructure, and manufacturing, including SCADA, PLC, and DCS technologies. They monitor and control physical assets in sectors like energy, water, and transportation to enhance reliability and efficiency.
- Supervisory Control and Data Acquisition (**SCADA**): Systems that monitor and control geographically dispersed assets.
- Distributed Control Systems (**DCS**): Control systems typically used in manufacturing environments to manage complex, interactive processes.
- Programmable Logic Controllers (**PLC**): Solid-state devices that use programmed logic to control machines and processes.
- **Field Devices**: Sensors, motors, relays, and actuators that perform the physical actions.
